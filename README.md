# qiangcai
修改版抢菜脚本，适用于新旧版本美团买菜</br>
新增支付会震动铃声提醒功能</br>
手机运行脚本建议使用Auto.js</br>
使用方法：</br>
1.打开下载好的Auto.js，点左上角菜单栏，打开无障碍服务，不同手机设置界面可能不一样，但都有《下载服务》栏</br>
2.点Auto.js，打开服务</br>
3.返回主界面，点右下角 + 号新建文件或者项目，把main.js的代码复制进去，或者直接把main.js导入进去</br>
4.点文件后面的三角运行按钮。</br>
注：一定要把定位权限修改权限都打开</br>
    建议美团买菜改成极速支付</br>
